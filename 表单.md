# 用户画像个性化推荐表单 - 需求逻辑

## 第一部分：基础信息 (Basic Info)
*用于建立用户基础模型，确定营养素剂量的基准权重*

1. **您的年龄段** (单选)
   - [ ] 18-30岁 (青年：关注效率、运动、颜值)
   - [ ] 31-50岁 (中年：关注压力管理、抗衰、体能)
   - [ ] 50岁以上 (中老年：关注心脑健康、骨骼、免疫)
2. **您的性别** (单选)
   - [ ] 男 (偏向：精力、肌肉、前列腺健康)
   - [ ] 女 (偏向：皮肤、贫血补铁、卵巢保养)

## 第二部分：生活状态与健康诉求 (Health Goals)
*多选题，每个选项对应后台一个或多个商品分类。用户勾选即增加对应分类的权重值*

- [ ] **经常熬夜，入睡困难或睡眠质量不佳**
  - *映射分类：[改善睡眠]*
- [ ] **长时间面对电脑/手机，感觉眼睛干涩、疲劳**
  - *映射分类：[视力护眼]*
- [ ] **有规律健身习惯，或近期计划增肌/减脂/提高耐力**
  - *映射分类：[运动营养]*
- [ ] **饮食不规律（外卖多、挑食），想补充日常营养缺失**
  - *映射分类：[综合维矿]*
- [ ] **关注血压、血脂指标，或关注心脑血管保养**
  - *映射分类：[心血管健康]*

## 第三部分：消费偏好与预算 (Budget Range)
*用于过滤搜索结果或排序，确保推荐符合用户消费能力*

1. **单件产品的月预算倾向** (单选)
   - [ ] **简约性价比 (¥100以内)**
   - [ ] **均衡品质感 (¥100 - ¥300)**
   - [ ] **高端专业级 (¥300以上)**

---

## 推荐逻辑简述 (给开发参考)
1. **分类匹配：** 根据第二部分勾选的健康诉求，确定主推分类。
2. **价位过滤：** 在主推分类中，筛选出价格在第三部分预算范围内的商品。
3. **优先级排序：** - 匹配度越高（满足多个诉求）的商品排在越前面。
   - 根据第一部分年龄/性别对特定成分（如：叶黄素、辅酶Q10）进行二次加权。
